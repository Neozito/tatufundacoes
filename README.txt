Projeto estático da landing page da Tatu Fundações.

Arquivos principais:
- index.html
- style.css
- script.js
- assets/

Funcionalidades:
- navegação one-page com seções Início, Quem Somos, Serviços e Contato
- botão flutuante de WhatsApp
- formulário que abre mensagem no WhatsApp
- mapa do Google Maps incorporado
- ícones em SVG locais
- imagens locais na pasta assets/images

Para publicar:
1. Envie todos os arquivos para a hospedagem
2. Garanta que o arquivo principal fique como index.html
3. Se desejar trocar imagens, substitua os arquivos em assets/images mantendo os mesmos nomes
